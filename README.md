# My Journey — AI & ML Specialist

Personal tracker for an AI & Machine Learning Specialist training program.

## Features
- Overall course progress in percentage
- Status per module: not started / in progress / completed
- Hours spent and final grade (18–30, with honors)
- Weighted average of grades on completed modules
- Personal notes per module
- Export / Import data as JSON to sync between devices

## Syncing Between Devices
Data is saved in the local browser. To transfer it:
1. **Export** the `.json` file from the source device
2. **Import** the file on the new device
